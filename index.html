Phantom List (Automation App)

Phantom List is a desktop application for automating a set of instruction for mouse and keyboard, programed in Pyhon and using PyQt5 framework and PyAutoGui module.


Technology Used

  • Python
  • PyQt5
  • PyAutoGui

Notable Features

  • Automation
  • Minimalist User-Interface
  • Save Files
  • Load Files

Difficulties

When running the program there is a continuous loop, which uploads the graphics onto the screen. An issue occurred when trying to run a second "while loop" for the countdown function, which resulted in the GUI to become none-responsive until the loop terminated.

Solutions

To overcome the looping issue, I had to implement threading, which is a module in Python that allows a second "while loop" to process separately.

Energy Consumption Calculator

The Energy Calculator is a text-based program written to help investigate customers' consumption and locate any negative reads. For both gas and electric, the calculation is activated when the user selects "export reports" to which the program generates a text file report for both energies.


Technology Used

  • Python
  • Pickle module

Notable Features

  • Pickle Module
  • Load Data
  • Save Data
  • Export via Csv
  • Calulating Energy Units
  • Delete Previous
  • Generates Report

Difficulties

Creating a text-based user interface that is clear and user-friendly.

Solutions

I took advantage of the "Sys" module in Python and applied "sys.clear()" as part of the "update" function to clear the screen and print out updated text. This resulted in a more user-friendly experience with clear structuring of text.
